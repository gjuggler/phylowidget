Step 1 of 10: PhyloWidget Gallery

Welcome to PhyloWidget!

This is the first in a series of short demonstrations or 'vignettes' designed to introduce the user to PhyloWidget, its capabilities, and how to use it effectively for phylogenetic visualization.

Shown on the left is PhyloWidget in its most basic configuration: no menus, no user interface, just displaying a tree. Try using some of the links below to sample the diversity of phylogenetic visualizations which PhyloWidget is capable of producing.

Tree gallery

  • Simple trees: letters, numbers, special characters
  • Test cases: single node, multifurcation, alphabet
  • Species trees: "tree of life", animals, archaea
  • Gene trees: ADH, Ensembl gene tree

Display options

PhyloWidget provides many options for how you display your trees. Below are some of the more commonly used options.

Note that any combination of options can be applied with any tree in the above list. Try playing around with different combinations of settings and trees!

  • Layouts: Rectangular, Diagonal, Circular, Unrooted
  • Branch lengths: show, ignore
  • Node size: tiny, medium, large
  • Line width: tiny, medium, large
  • Minimum text size: none, small, large
  • Text rotation: up, none, down
  • Angle quantization: none, quantize, level,
  • Take the next step!
